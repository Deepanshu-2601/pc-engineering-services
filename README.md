# PC Engineering Service - Corporate Website

A premium, fully-featured multi-page corporate website built for **PC Engineering Service**. This project transforms a single-page landing into a high-end enterprise platform mimicking the functionality of Aarvi Encon.

## 🌐 Live Demo
The website is hosted on GitHub Pages:
[https://Deepanshu-2601.github.io/pc-engineering-services/](https://Deepanshu-2601.github.io/pc-engineering-services/)

---

## 🔥 Key Features

### 1. Modern Multi-Page Architecture
- **Home**: Dynamic landing page with hero sections and service overviews.
- **About Us**: Dedicated page for company history (Est. 2019), Mission, Vision, and Leadership Team profiles.
- **Contact Us**: Professional contact portal with multiple office locations (Delhi & Mumbai) and a lead generation form.

### 2. Advanced UI/UX Elements
- **Interactive Mega Menus**: 3-column architectural dropdowns for Industries and Services.
- **Animated Statistics**: Real-time counter animation for Years of Excellence, Engineers Deployed, etc.
- **Clientele Grid**: Premium grid layout showcasing original logos of industry giants (L&T, Reliance, Tata, ONGC, and more).
- **Corporate Blog**: Clean grid layout for industry insights, engineering news, and company updates.

### 3. Responsive Design
- Optimized for Desktop, Tablet, and Mobile devices.
- High-performance glassmorphism effects and smooth micro-animations.

---

## 🛠️ Technology Stack
- **HTML5**: Semantic structure for SEO best practices.
- **Vanilla CSS3**: Custom design tokens, Flexbox/Grid layouts, and keyframe animations.
- **JavaScript (ES6+)**: Scroll-triggered animations, Intersection Observer API, and dynamic UI logic.
- **Tabler Icons**: Lightweight SVG icons for a modern architectural look.
- **Google Fonts**: "Inter" and "Outfit" typography for a premium feel.

---

## 🚀 Deployment
This site is automatically deployed via **GitHub Pages**.

To run locally:
1. Clone the repository.
2. Open `index.html` in any modern web browser.
